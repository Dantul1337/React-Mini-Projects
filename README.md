# React Frontend Projects ⚛️

Коллекция моих учебных мини-проектов на React, созданных для практики фронтенд-разработки.  
Каждый проект посвящён отдельной концепции или UI-паттерну.

## 🧠 Чему учусь через проекты
- **Хуки React** — useState, useEffect, useContext, useRef
- **Управление состоянием** — локальное состояние, контекст, простые редьюсеры
- **Работа с API** — GET/POST-запросы, индикация загрузки и обработка ошибок
- **TypeScript в React** — типизация props, событий, хуков
- **Сборка и деплой** — Vite, CRA, базовые настройки окружения

## 🛠️ Технологии
- React 18+
- TypeScript, JavaScript (ES6+)
- HTML5, CSS3
- REST API (публичные тестовые API)
